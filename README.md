# Sales Analysis Dashboard (Interactive Dashboard Creation Using Tableau)
## Project Objective
This project aims to perform data analysis using Tableau by creating dynamic and interactive dashboards. It focuses on uncovering sales trends, evaluating regional and product performance, and supporting data-driven decision-making.
## DataSet Used
-<a href="https://github.com/sabenaadham/Sales-Analysis-Dashboard/blob/main/Sales%20Data.xls">DataSet</a>

## Key Performance Indicators (KPIs)
- Total Sales Revenue
- Average Order Value (AOV)
- Sales Growth Rate
- Top-Selling Products
- Sales by Region
- Category Contribution
- Total Orders
- Peak Sales Periods
